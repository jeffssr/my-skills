# 证件照生成

将用户提供的照片转换为标准证件照。核心流程：解析要求 → 裁切宽高比 → 缩放像素 → 压缩文件大小。

## 标准尺寸（300 DPI）

| 名称 | 像素（宽×高） | 毫米 |
|------|--------------|------|
| 1寸 | 295×413 | 25×35 |
| 大1寸 | 390×567 | 33×48 |
| 小2寸 | 413×531 | 35×45 |
| 2寸 | 413×579 | 35×49 |
| 3寸 | 649×991 | 55×84 |

## 处理逻辑

1. **解析用户请求**：从自然语言中提取图片路径、目标尺寸、文件大小限制、输出格式
   - 未指定格式 → 默认 jpg
   - 未指定大小限制 → 默认 1MB
   - 未指定尺寸 → 必须询问用户

2. **验证图片路径**：确认文件存在且为图片格式

3. **调用脚本处理**：
   ```bash
   python "<skill-dir>/scripts/id_photo.py" "<图片路径>" --size "<尺寸名>" --max-size <MB数> --format <格式>
   ```
   自定义像素：
   ```bash
   python "<skill-dir>/scripts/id_photo.py" "<图片路径>" --width <宽> --height <高> --max-size <MB数> --format <格式>
   ```

4. **报告结果**：告知用户输出文件路径、像素尺寸、实际文件大小

## 用户意图识别

从用户描述中提取参数：

- 尺寸关键词："1寸"/"一寸"→ `--size 1寸`，"2寸"/"两寸"→ `--size 2寸`，以此类推
- "小2寸" → `--size 小2寸`，"大1寸" → `--size 大1寸`
- "不要超过500KB"/"500KB以内" → `--max-size 0.5`
- "PNG格式"/"png" → `--format png`
- "宽300高400像素" → `--width 300 --height 400`

## 输出

- 保存到原图同目录（除非用户指定其他路径）
- 命名规则：`原图文件名_尺寸.格式`（如 `photo_1寸.jpg`）
- 处理完成后向用户确认输出位置和文件大小

## 注意

- 需要 Pillow 库。如果未安装，先执行 `pip install Pillow`
- 裁切以图片中心为锚点，优先裁切宽度（左右）
- 二分查找 JPEG 质量参数，在文件大小限制内尽量保持高质量
